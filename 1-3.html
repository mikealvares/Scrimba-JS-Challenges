SCRIMBA - JSChallenge - First Duplicate


Given an array 'a' that contains only numbers, find the first duplicate number for which the second occurance has the minimal index. If there are more than 1 duplicated numbers retun the number for which the second occurance has a smaller index than the second occurance of the other number. IF no such element exists return -1
hint: hasownproperty()
eg:
  a = [2, 1, 3, 5, 3, 2]
  firstDuplicate(a) = 3

  a = [2, 4, 3, 1]
  firstDuplicate(a) = -1
